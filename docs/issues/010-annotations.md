## Goal

Create annotations that remain interoperable with major PDF viewers.

## Tasks

- [ ] Highlight, underline, and strikeout
- [ ] Sticky note and free text
- [ ] Rectangle, ellipse, line, and arrow
- [ ] Freehand ink
- [ ] Annotation property editor
- [ ] Flatten selected or all annotations

## Acceptance criteria

- Annotations display correctly in Acrobat Reader, Edge, and Chrome.
