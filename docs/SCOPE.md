# 機能スコープ

## 対象

### 閲覧

- PDFを開く、保存、別名保存
- 複数タブ
- 連続スクロール
- サムネイル
- ブックマーク
- ズーム、回転表示
- テキスト検索、コピー
- 印刷

### ページ整理

- 結合、分割、抽出
- 挿入、削除、複製、置換
- ドラッグ＆ドロップ並べ替え
- 回転、切り抜き
- 空白ページ、画像からPDF
- ページ番号、ヘッダー、フッター、透かし

### 注釈

- ハイライト、下線、取り消し線
- 付箋、フリーテキスト
- 線、矢印、矩形、楕円、フリーハンド
- スタンプ、署名画像
- 注釈のフラット化

### OCR・スキャン

- 日本語・英語OCR
- 透明テキスト層
- 傾き補正、自動回転
- 選択ページ・一括OCR
- Windows WIAスキャンは後続対応

### 墨消し・サニタイズ

- 矩形指定
- 検索語・正規表現指定
- 元テキスト・画像データの除去
- メタデータ、コメント、添付ファイル、JavaScriptの除去
- 墨消し結果の検証

### 圧縮・保護

- 画像ダウンサンプリング
- JPEG品質調整
- 不要オブジェクト削除
- AES暗号化
- 開封パスワード、権限設定

### フォーム

- AcroForm入力と保存
- フラット化
- 基本フィールド作成は後続対応

### 限定的な編集

- テキスト追加
- 短い既存テキストの削除・再描画による置換
- 画像追加、移動、サイズ変更、削除、置換
- リンク追加・編集

## 対象外

- クラウドストレージ
- 共同レビューとリアルタイム同期
- 他者への電子署名依頼・追跡
- PDF JavaScriptの実行
- XFAフォーム
- PDF Portfolio
- 3D、動画、音声コンテンツ
- 印刷業向けPreflight、色分解、トラッピング
- Word相当の完全な段落リフロー
- 完全なPDF/UAタグ編集
