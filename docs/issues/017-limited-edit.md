## Goal

Support practical short corrections without promising word-processor-style reflow.

## Tasks

- [ ] Add text with embedded fonts
- [ ] Remove and redraw short text regions
- [ ] Japanese font selection and metrics
- [ ] Add, move, resize, rotate, replace, and remove images
- [ ] Edit links

## Acceptance criteria

- Short Japanese and English corrections render consistently in major viewers.
- The UI clearly states that paragraph reflow is not supported.
