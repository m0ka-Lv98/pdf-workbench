## Goal

Create the stable UI shell used by all subsequent PDF tools.

## Tasks

- [ ] Multiple document tabs
- [ ] File open, close, recent files, and drag-and-drop
- [ ] Unsaved-change guard
- [ ] Status bar and error reporting
- [ ] Persist window geometry and preferences

## Acceptance criteria

- Multiple PDFs can be opened and closed independently.
- Closing a modified session requires explicit confirmation.
