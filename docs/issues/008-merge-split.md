## Goal

Cover the most common Acrobat page-management use cases.

## Tasks

- [ ] Merge multiple PDFs
- [ ] Split by page range and maximum pages
- [ ] Extract selected pages
- [ ] Create PDF from image files
- [ ] Preserve metadata and bookmarks according to explicit options

## Acceptance criteria

- Operations support large files without loading all page images into memory.
