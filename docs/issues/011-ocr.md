## Goal

Make scanned PDFs searchable without uploading them to a cloud service.

## Tasks

- [ ] Detect OCRmyPDF, Tesseract, and required external tools
- [ ] Select pages and languages
- [ ] Japanese and English presets
- [ ] Deskew and automatic rotation
- [ ] Progress, cancellation, and worker process isolation
- [ ] OCR output validation

## Acceptance criteria

- A scanned Japanese/English PDF becomes searchable and preserves page appearance.
